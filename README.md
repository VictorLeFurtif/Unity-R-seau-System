## CONCEPT
Inspiré du mode *Runners* de Garry's Mod, **Don't Get Caught** est un jeu multijoueur asymétrique opposant un Chasseur à plusieurs Souris. Le projet a été conçu pour tester les capacités de synchronisation réseau de **Unity NetCode** dans un environnement à haute vélocité.

---

## PHYSIQUE ET MOMENTUM
Le cœur du jeu réside dans son système de déplacement exigeant et nerveux, mettant l'accent sur la maîtrise du mouvement :
* **Conservation de l'inertie** : Chaque mouvement fluide augmente et maintient la vitesse du joueur.
* **Wall Jump & Momentum** : Les joueurs peuvent enchaîner les sauts contre les murs pour atteindre des zones inaccessibles ou semer le poursuivant.
* **Gameplay Aérien** : Le Chasseur est équipé d'un **Grappin** pour compenser la mobilité des souris et réduire les distances instantanément.

---

## MECANIQUES MULTIJOUEUR
Le jeu repose sur un cycle de poursuite dynamique :
* **Rôles** : Le Chasseur doit capturer toutes les Souris avant la fin du chronomètre.
* **Capture et Libération** : Une souris attrapée est envoyée en prison. Les joueurs encore libres peuvent prendre le risque de s'approcher de la zone de détention pour libérer leurs alliés.
* **Wall Hack (Chasseur)** : Pour éviter les parties trop longues, le Chasseur reçoit une vision à travers les murs à intervalle régulier pour localiser ses proies.
